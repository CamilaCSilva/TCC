<h1 align="center"> NODE VERSION MANAGER </h1>

> Obs.: Para instalar o NVM deve-se primeiramente desinstalar e remover o NPM e o NODE atual da maquina.

## Como instalar o NVM no Windows

Acesse o site do [NVM](https://github.com/coreybutler/nvm-windows/releases)  no Github e clique em `nvm-setup.zip`

<img src=https://github.com/CamilaCSilva/tcc/assets/85804680/6335267d-7dbd-4a92-8289-d71d5d219f0c width="700" >

Após executar o arquivo instalado do site os seguintes passo:

- Descompacta o arquivo zip e executa o `nvm-setup.exe`
  
  <img src=https://github.com/CamilaCSilva/tcc/assets/85804680/44957707-6820-4d98-bdd6-07a1b7b258ed width="700" >

- Aceite que o aplicativo faça alteração e os termos, após isso clique em `Next`
  
  <img src=https://github.com/CamilaCSilva/tcc/assets/85804680/60b3a51f-bc5d-437e-ba07-2057abed192f width="700" >
  
-  Após isso clique em `Next`
   
   <img src=https://github.com/CamilaCSilva/tcc/assets/85804680/6b259846-95ed-4a1d-bdb3-a41e706c3973 width="700" >
   
-  Após isso clique em `Next`
   
   <img src=https://github.com/CamilaCSilva/tcc/assets/85804680/b5dfb485-5928-47d8-a61b-bdf4ac560420 width="700" >
   
-  Após isso clique em `Install`
   
   <img src=https://github.com/CamilaCSilva/tcc/assets/85804680/b180c9a6-0250-44ed-8873-6d768238b4d5 width="700" >
   
-  Após isso clique em `Finish`
   
   <img src=https://github.com/CamilaCSilva/tcc/assets/85804680/cc958a7f-7ff0-43fa-9bc9-eafcff11f204 width="700" >


-  Agora vamos verificar se instalou corretamente e instalar a versão(16.16.0) do NODE para rodar a aplicação feita em Angular
   - Abrar o `CMD` ➡️ <kbd>Win</kbd> ➡️ digite: `CMD` ➡️ <kbd>Enter</kbd>
   - Digite os seguintes comandos
     ```
      nvm --version
      nvm install v16.16.0
      nvm use v16.16.0 (caso haja uma perguntar clique em SIM)
      node -v
     ```
   - Se tudo estiver certo vai aparecer conforme abaixo:
     
     <img src=https://github.com/CamilaCSilva/tcc/assets/85804680/a301c8f2-23e2-46ee-a05c-7abfa1a911cf width="700" >
